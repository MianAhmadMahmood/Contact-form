# Contact Form with JavaScript Validation

## 📌 Project Overview

This project was created as part of the **Vortex Tech Web Development Internship – Week 2** assignment.

The website includes a responsive contact form with JavaScript validation and a Dark/Light mode toggle.

---

## 🚀 Features

- Responsive Contact Form
- Name Validation
- Email Validation using Regular Expression (Regex)
- Message Validation
- Success Message without Page Reload
- Dark Mode / Light Mode Toggle
- JavaScript Event Handling using `addEventListener`
- Separate HTML, CSS, and JavaScript files

---


## 🛠 Technologies Used

- HTML5

---

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your web browser.
4. Fill out the contact form and test the validation.
5. Click the **Dark Mode** button to switch themes.
